## Prerequisites
- Python
- Code Editor (VScode, etc)


## Installation
1. Clone the repo
  ```sh
  git clone https://github.com/ARF-DEV/Defense-of-the-Ila.git
  ```
2. Install pygame
  ```sh
  pip install pygame 
  ```
3. Goto `src` folder
4. Launch `main.py` file
  ```sh
  python main.py 
  ```
